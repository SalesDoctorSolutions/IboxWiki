# Ежедневный отчет по клиентам

Отчет **Ежедневный отчет по клиентам** показывает, какие операции клиент совершил в течение дня, на какую сумму, а также его баланс на начало и конец дня.

Используйте отчет, чтобы:

- Контролировать движение средств по клиентам  
- Проверять проведенные документы за день  
- Отслеживать изменения баланса  

## Как открыть отчет

1. Откройте раздел **Отчеты** в левой панели.
2. В блоке **Отчеты по продажам** выберите **Ежедневный отчет по клиентам**.

По умолчанию открывается страница отчета с данными за текущий день.

## Доступные поля

В отчете можно отобразить следующие поля:

- **Клиент** — наименование клиента  
- **Дата** — дата формирования отчета  
- **Баланс на начало дня** — задолженность или переплата на начало дня  
- **Документы** — список проведенных за день документов  
- **Баланс на конец дня** — задолженность или переплата на конец дня
