Функция **Поиск по коду клиента или поставщика** помогает быстрее находить контрагентов в системе по уникальному коду.

Вы можете указать код при создании или редактировании клиента или поставщика. После этого код можно использовать для поиска:

- в списке контрагентов;
- при создании документов продаж, закупок и оплат.

Это ускоряет работу и снижает вероятность ошибок при выборе контрагента.

---

:::warning Важно
Код контрагента должен быть уникальным.

Система не позволит сохранить одинаковый код для разных клиентов или поставщиков.
:::

---

## Когда полезно использовать поиск по коду

Функция особенно полезна, если:

- в базе большое количество клиентов или поставщиков;
- в компании используются внутренние коды контрагентов;
- сотрудники работают с бумажными документами или внешними системами, где указан код;
- важно ускорить поиск контрагента при оформлении документов;
- требуется минимизировать ошибки при выборе клиента или поставщика.

## Как включить функцию

1. Откройте **Настройки** на левой панели.  
2. Перейдите в раздел **Настройки организации**.  
3. Откройте вкладку **Функционал**.  
4. Найдите параметр **Поиск по коду клиента или поставщика** и нажмите на иконку **карандаша**.  
5. Выберите значение **Да**.  
6. Нажмите на **галочку**, чтобы сохранить изменения.

## Как указать код для контрагента

### Указать код для клиента

1. Откройте раздел **Клиенты → Клиенты**.  
2. Создайте нового клиента или откройте существующего для редактирования.  
3. Перейдите на вкладку **Прочее**.  
4. Укажите уникальный код в поле **Код**.  
5. Нажмите **Сохранить**.

### Указать код для поставщика

1. Откройте раздел **Поставщики → Поставщики**.  
2. Создайте нового поставщика или откройте существующего для редактирования.  
3. Перейдите на вкладку **Прочее**.  
4. Укажите уникальный код в поле **Код**.  
5. Нажмите **Сохранить**.
