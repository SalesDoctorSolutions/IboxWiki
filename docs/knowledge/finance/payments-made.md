# Расходы

---

## Как вносить оплату поставщикам
Чтобы внести оплату поставщику:

1. Перейдите в раздел **Деньги → Расходы**.
2. Нажмите на **плюс** в правом верхнем углу.
3. Выберите опцию **Оплата поставщику**.
4. Выберите поставщика.
5. Заполните данные оплаты:
   - сумму;
   - валюту;
   - кассу  
   (при необходимости добавьте ещё кассы).
6. Нажмите **Сохранить**.

---

## Как возвращать деньги клиентам
При получении возврата от клиента необходимо оформить возврат денег клиенту:

1. Перейдите в раздел **Деньги → Расходы**.
2. Нажмите на **плюс** в правом верхнем углу.
3. Выберите опцию **Возврат денег клиенту**.
4. В открывшейся форме укажите клиента.
5. Заполните данные оплаты:
   - сумму;
   - валюту;
   - кассу  
   (при необходимости добавьте ещё кассы).
6. Нажмите **Сохранить**.

---

## Как оформлять прочие расходы
С помощью прочих расходов вы можете вносить в программу расходы, не связанные с закупкой товара, например: ремонт, аренда помещения или налоги.

Для этого используйте **прочие расходы**:

1. Перейдите в раздел **Деньги → Расходы**.
2. Нажмите на **плюс** в правом верхнем углу.
3. Выберите опцию **Прочий расход**.
4. В открывшейся форме выберите тип расхода  
   (при необходимости можно создать новый тип).
5. Заполните данные оплаты:
   - сумму;
   - валюту;
   - кассу  
   (при необходимости добавьте ещё кассы).
6. Нажмите **Сохранить**.

:::tip **Совет:**  
Привязывайте типы расходов к категориям расходов, чтобы анализировать сумму операционных расходов по каждой из категорий.
:::
