# API и интеграции iBox

iBox предоставляет API для интеграции с внешними системами: сайтами, CRM, мобильными приложениями и другими сервисами.

С помощью API вы можете:

- получать данные о товарах, клиентах и документах;
- создавать заказы, отгрузки и другие операции;
- автоматизировать бизнес-процессы;
- интегрировать iBox с вашими внутренними системами.

---

## Как получить API-ключ

API-ключ используется для авторизации при работе с API.

Создать ключ может только пользователь с правами администратора.

### Шаги

1. Нажмите на **название организации** в верхней панели.  
2. Выберите **Управление**.  
3. В левой панели перейдите в раздел **Настройки**.  
4. Откройте раздел **Ключи API**.  
5. Нажмите **Плюс**.  
6. Заполните:
   - **Название ключа** — для удобства идентификации;  
   - **Организации** — выберите, к каким организациям будет доступ.  
7. Нажмите **Сохранить**.  
8. Скопируйте сгенерированный ключ.

:::warning Важно
Сохраните API-ключ сразу после создания.

После закрытия окна вы **не сможете** посмотреть его повторно.  
Если ключ утерян — потребуется создать новый.
:::

## Управление API-ключами

### Как деактивировать ключ

Если ключ больше не используется или требуется ограничить доступ:

1. Откройте **Управление → Настройки → Ключи API**.  
2. Найдите нужный ключ.  
3. Нажмите на меню **⋮ (три точки)** справа.  
4. Выберите **Отключить**.

После деактивации ключ перестанет работать.

---

## Рекомендации по безопасности

- не передавайте API-ключ третьим лицам;
- храните ключи в защищенном месте (например, в переменных окружения);
- используйте отдельные ключи для разных интеграций;
- при подозрении на компрометацию немедленно отключите ключ и создайте новый.
