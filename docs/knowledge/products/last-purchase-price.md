# Последняя закупочная цена

**Последняя закупочная цена** — это цена, по которой товар был закуплен в последнем приходном документе.

Она автоматически обновляется при каждом новом поступлении товара и используется системой как актуальная ориентировочная себестоимость.

:::info Информация
Если в настройках не указан прайс-лист по умолчанию для закупок, при поступлении товаров система автоматически будет заполнять именно последнюю закупочную цену.
:::

---

## Для чего используется

Последняя закупочная цена помогает:

- быстро ориентироваться в актуальной себестоимости товара;
- контролировать маржинальность при продажах;
- использовать цену как базу для расчета продажных цен;
- анализировать изменения закупочных цен со временем.

---

## Где посмотреть последнюю закупочную цену

1. Перейдите в раздел **Товары → Товары**.  
2. Откройте нужный товар.  
3. Прокрутите вниз до блока цен.  

Здесь отображается последняя закупочная цена.

---

## Использование в продажах

По умолчанию последняя закупочная цена применяется только в закупках и корректировках.

При необходимости вы можете разрешить ее использование в продажах.

### Как включить

1. Откройте **Настройки**.  
2. Перейдите в раздел **Настройки организации**.  
3. Откройте вкладку **Функционал**.  
4. Найдите параметр **Использовать последнюю закупочную цену в продажах**.  
5. Выберите значение **Да**.  
6. Сохраните изменения.

### Когда это полезно

- при быстром оформлении продаж без заранее заданных цен;
- если цены формируются от закупочной стоимости;
- при работе с переменной наценкой;
- для контроля минимальной цены продажи.

---

## Критерий обновления цены

Вы можете выбрать, как определяется последняя закупочная цена:

- по дате документа;
- по дате создания документа.

### Как настроить

1. Откройте **Настройки**.  
2. Перейдите в раздел **Настройки организации**.  
3. Откройте вкладку **Функционал**.  
4. Найдите параметр **Критерий обновления последней закупочной цены**.  
5. Выберите нужную опцию.  
6. Сохраните изменения.

### Когда какую опцию использовать

- **По дате документа** — если документы вводятся задним числом и важна фактическая дата закупки;  
- **По дате создания** — если важен порядок ввода документов в систему и исключаются изменения «задним числом».  

---

## Влияние корректировок

По умолчанию положительные корректировки (увеличение остатков) обновляют последнюю закупочную цену.

При необходимости это можно отключить:

1. Откройте **Настройки**.  
2. Перейдите в раздел **Настройки организации → Функционал**.  
3. Найдите параметр **Разрешить корректировкам остатков изменять последнюю закупочную цену**.  
4. Установите значение **Нет**.  
5. Сохраните изменения.

---

## Межорганизационные приемки

По умолчанию приемка товаров между организациями также обновляет последнюю закупочную цену.

Чтобы отключить:

1. Откройте **Настройки**.  
2. Перейдите в раздел **Настройки организации → Функционал**.  
3. Найдите параметр **Разрешить межорганизационным приемкам изменять последнюю закупочную цену**.  
4. Установите значение **Нет**.  
5. Сохраните изменения.

---

:::warning Важно

При изменении параметров или критериев обновления цена **не пересчитывается автоматически**.

Чтобы обновить последнюю закупочную цену, необходимо создать новый документ поступления с данным товаром.
:::
