# Использовать НДС

Данная функция позволяет **учитывать НДС для каждого товара** и использовать эти данные при оформлении документов и печати накладных.

После включения функции:

- в таблице товаров в документах **отгрузки, заказов, возвратов, закупок, заказов поставщикам и возвратов поставщикам** появится поле для указания **процента НДС**;
- в карточке товара во вкладке **Прочее** появится возможность задать **НДС по умолчанию**.

Функция может использоваться при печати накладных, чтобы отображать **процент НДС и цену без НДС**.

---

:::info Информация
В iBox по умолчанию **цены указываются с учетом НДС**.
:::

---

## Когда полезно использовать функцию

Использование функции помогает:

- корректно оформлять документы с учетом налоговых требований;
- показывать клиентам и поставщикам сумму и процент НДС в накладных;
- автоматически подставлять НДС в документы по каждому товару;
- уменьшать количество ошибок при ручном расчете налогов;
- вести более прозрачный учет продаж и закупок.

Это особенно полезно, если:

- компания является плательщиком НДС;
- требуется печать официальных накладных;
- товары облагаются разными ставками НДС;
- важно видеть цену товара без НДС.

---

## Как включить функцию

1. Перейдите в **Настройки** на левой панели.
2. В настройках организации откройте раздел **Настройки**.
3. Перейдите в раздел **Функционал**.
4. Найдите параметр **«Использовать НДС»** и нажмите на иконку **карандаша** справа.
5. Выберите опцию **Да**.
6. Нажмите на **синюю иконку галочки**, чтобы сохранить изменения.

---

## Как указать процент НДС по умолчанию для товара

1. Перейдите в раздел **Товары → Товары**.
2. Наведите курсор на правую часть нужного товара и нажмите на иконку ✏️ **Карандаш** (**Редактировать**).
3. Откройте вкладку **Прочее**.
4. Укажите **процент НДС** для товара.
5. Нажмите **Сохранить**.

---

## Как добавить НДС в печатную накладную

1. Перейдите в **Настройки** на левой панели.
2. В настройках организации откройте раздел **Шаблоны для печати**.
3. Скачайте нужный шаблон и отредактируйте его.
4. При редактировании файла Excel добавьте в таблицу товаров необходимые значения:
   - **Процент НДС** — `[products.vat_percent]`
   - **Цена без НДС** — `[products.origin_price]`
5. В разделе **Шаблоны для печати** нажмите **Загрузить шаблон**.
6. Выберите сохраненный файл Excel.
7. Нажмите **Сохранить**.
