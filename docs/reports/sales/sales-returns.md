## Возвраты покупателей

Отчет **Возвраты покупателей** показывает количество и сумму возвратов по каждому товару за выбранный период.

Используйте отчет, чтобы:

- Анализировать объем возвратов
- Контролировать сумму возвратов
- Выявлять товары с наибольшим количеством возвратов

## Как открыть отчет

1. Откройте раздел **Отчеты** в левой панели.
2. В блоке **Отчеты по продажам** выберите **Возвраты покупателей**.

Откроется страница отчета с данными за выбранный период.

## Доступные поля

В отчете можно отобразить следующие поля:

- **Товар** — наименование товара
- **Количество** — количество возвращенных единиц
- **Сумма** — общая сумма возвратов по товару
