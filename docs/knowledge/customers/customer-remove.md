# Удаление и деактивация клиентов

В iBox есть несколько способов убрать клиента из списков: удалить его, деактивировать или добавить в чёрный список.


### Как удалить клиента
Удаление используется только в том случае, если клиент был создан по ошибке.  
Если с клиентом уже есть история, система не позволит его удалить. Очистка истории может привести к ошибкам в отчётах.

Чтобы удалить клиента:
1. Перейдите в раздел **Клиенты**.
2. Наведите курсор на правую часть строки нужного клиента.
3. Нажмите на иконку 🗑 **корзины**.
4. Подтвердите удаление.

---

### Как деактивировать клиента
Если с клиентом есть история, но вы больше не работаете с ним, его можно деактивировать.  
Клиент исчезнет из рабочих списков, но сохранится в отчётах.

Чтобы деактивировать клиента:
1. Перейдите в раздел **Клиенты**.
2. Выберите одного или несколько клиентов.
3. Нажмите на три точки в верхней части экрана.
4. Выберите опцию **Сделать неактивным**.

:::warning **Важно**  
Если нужно убрать клиента только из одной организации, но продолжать использовать его в других, настройте организации клиента через панель администратора.
:::
---

### Чёрный список
Если вы не хотите, чтобы клиент покупал у вас товары, но хотите принимать от него оплату, добавьте его в чёрный список:

1. Перейдите в раздел **Клиенты**.
2. Наведите курсор на правую часть строки выбранного клиента.
3. Нажмите на иконку ✏️ **карандаша** (редактировать).
4. Откройте вкладку **Прочее**.
5. Поставьте галочку **В чёрном списке**.

---

### Что выбрать

| Ситуация | Действие |
|---------|----------|
| Клиент создан по ошибке | Удаление клиента |
| Не работаете с клиентом в одной организации, но используете в других | Настройка организаций клиента |
| Больше не работаете с клиентом, но у него есть история | Деактивация клиента |
| Не хотите продавать клиенту, но хотите принимать оплату | Чёрный список |
