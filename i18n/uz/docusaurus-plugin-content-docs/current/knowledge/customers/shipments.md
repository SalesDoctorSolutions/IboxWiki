# Sotuvlar

Sotuvlar mahsulotning mijozga topshirilganini qayd etadi — aynan shu bosqichdan debitorlik qarzini nazorat qilish va to‘lovlar bilan ishlash boshlanadi.

iBOX tizimida sotuvlar savdo jarayonining asosiy hujjatlaridan biridir.

---

## Sotuvlarni qanday rasmiylashtirish mumkin

Sotuv yaratish uchun:

1. **Sotuvlar** modulidagi **Sotuvlar** bo‘limiga o‘ting.
2. Yuqori o‘ng burchakdagi **plyus** belgisini bosing.
3. Mijozni tanlang.
4. Hujjat valyutasini tanlang.
5. Mahsulotlar sotaladigan **omborni** tanlang.
6. Mahsulotlarni ro‘yxatga qo‘shing:
   - qo‘lda mahsulotlarni qidirish orqali;
   - mahsulotlarni tanlash orqali;
   - Excel’dan yuklash orqali.
7. Ma’lumotlarni tekshiring:
   - miqdor;
   - narxlar.
8. **Saqlash** tugmasini bosing.

Sotuv vaqtida qo‘shimcha ravishda quyidagilarni belgilashingiz mumkin:
- vositachi;
- haydovchi;
- to'lash muddati;
- to‘lov usuli;
- izoh;
- fayllarni biriktirish.

---

## Jo‘natmani tahrirlash

Agar o‘zgartirish yoki tuzatish kiritish kerak bo‘lsa:

1. **Sotuvlar** modulidagi **Sotuvlar** bo‘limiga o‘ting.
2. Kerakli Sotuvlar qatorining o‘ng tomoniga kursorni olib boring.
3. ✏️ **Qalam** ikonkasini bosing (Tahrirlash).
4. O‘zgartirishlarni kiriting.
5. **Saqlash** tugmasini bosing.

---

## Yuk xatini chop etish

1. **Sotuvlar** bo‘limida kerakli Sotuvlar qatorining o‘ng tomoniga kursorni olib boring.
2. **Printer** ikonkasini bosing.
3. Oldindan ko‘rish oynasida printerni tanlang va sozlamalarni tekshiring.
4. Chop etishni boshlash uchun **Print** tugmasini bosing.

---

## Sotuvlar bo‘yicha to‘lovni qabul qilish

### Sotuv vaqtida to‘lov

Siz Sotuvni yaratish jarayonida darhol to‘lovni qabul qilishingiz mumkin.

:::warning **Muhim**  
Buning uchun **Sozlamalar → Funksional** bo‘limida  
**To‘lovni hujjatlar bo‘yicha taqsimlash** funksiyasini yoqing.
:::

1. Sotuv ma’lumotlarini to‘ldiring — mijoz, mahsulotlar, narxlar.
2. Sahifani pastga aylantirib, **To‘lov qo‘shish** tugmasini bosing.
3. Quyidagilarni to'ldiring:
   - summa;
   - valyuta;
   - kassa.
4. **Saqlash** tugmasini bosing.
5. Sotuvni yakunlang.

### Sotuvdan keyin to‘lov

Sotuv yakunlangandan so‘ng ham to‘lovni qabul qilish mumkin:

1. **Sotuvlar** bo‘limiga o‘ting.
2. Kerakli Sotuvning **raqami** ustiga bosing.
3. Yuqori o‘ng burchakda **To'lov kiritish** tugmasini bosing.
4. Quyidagilarni kiriting:
   - summa;
   - valyuta;
   - kassa.
5. **Saqlash** tugmasini bosing.
