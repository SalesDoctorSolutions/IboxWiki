# Возможность создавать сетевых покупателей

Данная функция позволяет создавать **сетевых клиентов и поставщиков**, состоящих из **головного офиса и филиалов**.

Все филиалы объединяются в одну сеть, при этом:

- **взаиморасчеты ведутся только с головным офисом**;
- **акт сверки формируется по головному офису**;
- все продажи филиалам фиксируются как задолженность головного офиса;
- все оплаты принимаются от имени головного офиса;
- операции филиалов автоматически учитываются в общем балансе сети.

---

:::info Информация
В iBox у каждого контрагента можно указать только **один адрес**.  
Если у клиента или поставщика несколько торговых точек или объектов — используйте функцию сетевых покупателей.
:::

---

:::warning Важно
Несмотря на общий баланс сети, **баланс кешбэка ведется отдельно по каждому филиалу**.  

Каждый филиал может использовать **только свой накопленный кешбэк**.
:::

---

## Когда полезно использовать сетевых покупателей

Функция особенно полезна, если:

- клиент имеет несколько магазинов или филиалов;
- поставщик работает через региональные представительства;
- необходимо вести единый баланс по всей сети;
- требуется формировать общий акт сверки;
- нужно анализировать продажи по отдельным точкам клиента;
- важно упростить финансовый учет и контроль задолженности.

---

## Как включить функцию

1. Перейдите в **Настройки** на левой панели.
2. В настройках организации откройте раздел **Настройки**.
3. Перейдите в раздел **Функционал**.
4. Найдите параметр **«Использовать сетевых покупателей и поставщиков»** и нажмите на иконку **карандаша** справа.
5. Выберите опцию **Да**.
6. Нажмите на **синюю иконку галочки**, чтобы сохранить изменения.

---

## Как создать головной офис сетевого клиента

1. Перейдите в раздел **Клиенты → Клиенты**.
2. Нажмите **плюс** в правом верхнем углу.
3. В форме создания включите опцию **«Входит в сеть»**.
4. Выберите вариант **«Это головной офис»**.
5. Заполните данные:
   - название;
   - контактные данные;
   - адрес;
   - реквизиты.
6. Нажмите **Сохранить**.

---

:::info Информация
Официальное название контрагента указывается **у головного офиса** во вкладке **Реквизиты**.  

Все филиалы автоматически используют такое же официальное название.
:::

---

## Как создать филиал сетевого клиента

1. Перейдите в раздел **Клиенты → Клиенты**.
2. Нажмите **плюс** в правом верхнем углу.
3. Включите опцию **«Входит в сеть»**.
4. Выберите вариант **«Это филиал»**.
5. Выберите **головной офис** из списка.
6. Заполните данные:
   - название;
   - контактные данные;
   - адрес.
7. Нажмите **Сохранить**.

---

:::info Информация
По аналогии можно создавать **сетевых поставщиков**, чтобы объединять филиалы поставщика в одну сеть с единым балансом взаиморасчетов.
:::
