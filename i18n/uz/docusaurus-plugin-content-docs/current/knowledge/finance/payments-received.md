# To‘lov qo‘shish

## Mijozlardan to‘lov qabul qilish
Mijozdan to‘lov qo‘shish uchun:

1. **Pul → Kirimlar** bo‘limiga o‘ting  
2. O‘ng yuqori burchakdagi **plus** tugmasini bosing  
3. **Mijozdan to‘lov** variantini tanlang  
4. Mijozni tanlang  
5. To‘lov ma’lumotlarini kiriting:
   - summa  
   - valyuta  
   - kassa  
   (zarurat bo‘lsa qo‘shimcha kassalar qo‘shishingiz mumkin)  
6. **Saqlash** tugmasini bosing  

:::info **Ma’lumot:**  
Mijozdan to‘lovni sotuv jarayonida yoki **Qarzdorlar** bo‘limi orqali ham qo‘shish mumkin.
:::

---

## Ta'minotchidan pul qaytarib olish

Ta'minotchiga qaytarish rasmiylashtirilgandan so‘ng yoki oldindan to‘lov kiritilib, majburiyat bajarilmaganda pulni qaytarib olish zarur bo‘ladi.

Qaytarib olish uchun:

1. **Pul → Kirimlar** bo‘limiga o‘ting  
2. **plus** tugmasini bosing  
3. **YTa'minotchidan qaytgan pul** variantini tanlang  
4. Ta'minotchini belgilang  
5. To‘lov ma’lumotlarini kiriting:
   - summa  
   - valyuta  
   - kassa  
   (zarurat bo‘lsa qo‘shimcha kassalar qo‘shing)  
6. **Saqlash** tugmasini bosing  

---

## Boshqa kirimlar

Mijozlar yoki ta'minotchilar bilan bog‘liq bo‘lmagan pul kirimlarni ham kiritish mumkin.  
Masalan, shaxsiy mablag‘ingizni kassaga qo‘shganingizda.

Buning uchun **boshqa kirimlar**dan foydalaning:

1. **Pul → Kirimlar** bo‘limiga o‘ting  
2. **plus** tugmasini bosing  
3. **Boshqa kirimlar** variantini tanlang  
4. Kirim turini tanlang  
   (zarurat bo‘lsa yangi tur yaratish mumkin)  
5. To‘lov ma’lumotlarini kiriting:
   - summa  
   - valyuta  
   - kassa  
   (zarurat bo‘lsa qo‘shimcha kassalar qo‘shing)  
6. **Saqlash** tugmasini bosing
