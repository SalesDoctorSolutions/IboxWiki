# Добавление базовых элементов

## Как добавить кассы

Добавление касс необходимо для разделения балансов по разным способам оплаты  
(например, наличная касса, терминал или перечисление), сотрудникам или другим признакам.

Чтобы добавить кассу:

1. Нажмите **Настройки** в нижней части боковой панели.
2. В настройках организации перейдите в раздел **Кассы**.
3. В правом верхнем углу нажмите **+ Касса**.
4. Укажите:
   - название;
   - способ оплаты;
   - сортировочный номер.
5. Нажмите **Сохранить**.

## Как добавить склады

Добавление складов необходимо для разделения остатков по разным локациям,  
типам товаров (например, сырье и готовая продукция), сотрудникам или другим признакам.

Чтобы добавить склад:

1. Нажмите **Настройки** в нижней части боковой панели.
2. В настройках организации перейдите в раздел **Склады**.
3. В правом верхнем углу нажмите **+ Склад**.
4. Укажите:
   - название;
   - адрес;
   - сортировочный номер.
5. Нажмите **Сохранить**.
