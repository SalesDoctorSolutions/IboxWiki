# Управление финансами

## Как просматривать остатки денег в кассах

Чтобы посмотреть подробный отчет по деньгам в кассах по каждой валюте:

1. Перейдите в раздел **Деньги → Остатки денег**.
2. В отчете отобразятся остатки по каждой кассе и валюте.

### Детализация операций

- Нажмите на сумму в нужной кассе.
- Откроется детальный список всех операций по выбранной кассе и валюте.

## Перемещения денег

Перемещения денег в iBOX используются для учета переноса средств с одной кассы на другую.

### Как оформить перемещение денег

1. Перейдите в раздел **Деньги → Перемещения**.
2. Нажмите кнопку **Перемещение денег** в правом верхнем углу.
3. Выберите опцию:
   - **Создать** — для одного перемещения;
   - **Создать несколько** — если нужно оформить несколько перемещений одновременно.
4. Заполните данные:
   - **Из кассы**;
   - **В кассу**;
   - **Сумма**;
   - **Валюта**.
5. Нажмите **Сохранить**.

## Конвертации валют

В iBOX каждая валюта в кассе хранится раздельно.

Если вы обменяли валюту в своей кассе и хотите отразить это в системе, используйте функцию **Конвертации валют**.

### Как оформить обмен валюты

1. Перейдите в раздел **Деньги → Конвертации**.
2. Нажмите кнопку **+ Обмен валюты** в правом верхнем углу.
3. Заполните данные:
   - **Касса**;
   - **С какой валюты**;
   - **На какую валюту**;
   - **Сумма**;
   - **Курс**.
4. Нажмите **Сохранить**.
