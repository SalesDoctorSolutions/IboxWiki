# Sotuvlar

**Sotuvlar** hisoboti barcha sotuvlar va qaytishlarning batafsil ro‘yxatini har bir mahsulot bo‘yicha alohida ko‘rsatadi.

:::info Ma’lumot
Faqat qaytishlarni ko‘rish uchun **Hujjat turi** filtrini sozlang.  
Hisobotda qaytishlar qizil rang bilan ajratib ko‘rsatiladi.
:::

Hisobotdan quyidagi maqsadlarda foydalanish mumkin:

- Sotilgan mahsulotlar hajmini nazorat qilish
- Sotuvlar bo‘yicha daromadni tahlil qilish
- Qo‘llanilgan narxlar va chegirmalarni tekshirish

## Hisobotni qanday ochish mumkin

1. Chap panelda **Hisobotlar** bo‘limini oching.
2. **Savdo hisobotlari** ustunida **Sotuvlar** hisobotini tanlang.

Tanlangan davr bo‘yicha ma’lumotlar ko‘rsatilgan hisobot sahifasi ochiladi.

## Mavjud ustunlar

Hisobotda quyidagi ustunlarni ko‘rish mumkin:

- **Mahsulot** — sotilgan mahsulot nomi  
- **Hujjat sanasi** — hujjat yaratilgan sana  
- **Hujjat soni / Raqami** — hujjat raqami yoki guruhlashda hujjatlar soni  
- **Mijoz soni / Nomi** — mijoz nomi yoki guruhlashda mijozlar soni  
- **Mas’ul shaxs** — hujjat uchun mas’ul shaxs  
- **Miqdor** — hujjatdagi mahsulot miqdori  
- **Narx** — mahsulotning birlik narxi  
- **Summa** — pozitsiya bo‘yicha yakuniy summa  
- **Artikul** — mahsulot kodi  
- **Hujjat turi** — yuk jo‘natish yoki qaytarish  
- **Vositachi** — hamkor yoki agent  
- **Chegirma foizda** — chegirma miqdori foizlarda  
- **Chegirma** — pozitsiya bo‘yicha chegirma summasi  
- **Jami chegirma** — pozitsiya bo‘yicha umumiy chegirma summasi  
- **Chegirmadan oldingi narx** — chegirma hisobga olinmagan narx  
- **Chegirmadan oldingi jami** — chegirma hisobga olinmagan summa  
- **Kategoriya** — mahsulot kategoriyasi  
- **Brend** — mahsulotning savdo belgisi  
- **Prays-list** — hujjatda qo‘llangan prays-list
