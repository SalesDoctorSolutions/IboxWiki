# Категории клиентов

**Категория клиентов** — это инструмент для группировки клиентов в системе.

Используйте категории, чтобы разделять клиентов по типу, направлению деятельности, региону или любому другому признаку, который важен для вашего бизнеса.  

Категории помогают структурировать клиентскую базу и упрощают работу с ней.

---

## Как добавить категорию клиентов
1. Нажмите **Настройки** в левой панели.
2. Пролистайте вниз и в разделе **Общие настройки** перейдите в **Категории клиента**.
3. Нажмите **+ Категория клиента** в правом верхнем углу.
4. Заполните данные:
   - название;
   - сортировочный номер.
5. Нажмите **Сохранить**.
