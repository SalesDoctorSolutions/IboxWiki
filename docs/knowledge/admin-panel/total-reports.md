# Общие отчёты по всем организациям

## Отгрузки по всем организациям
Вы можете просматривать подробный список отгрузок по всем организациям:

1. Перейдите в **панель администратора**.
2. Откройте раздел **Отчёты → Отгрузки**.

---

## Остатки по всем организациям
Вы можете просматривать остатки товаров по всем организациям и складам:

1. Перейдите в **панель администратора**.
2. Откройте раздел **Отчёты → Остатки**.

Здесь отображается общее количество товаров во всех организациях.  
Чтобы увидеть остатки подробно по каждой организации и складу, нажмите на название товара.

---

## Финансовые операции по всем организациям
Вы можете просматривать подробный список денежных операций по всем организациям:

1. Перейдите в **панель администратора**.
2. Откройте раздел **Отчёты → Финансовые операции**.
