# Значения по умолчанию

Чтобы не вводить одни и те же данные при создании каждого документа, можно задать значения по умолчанию.  
Тогда нужные поля будут заполняться автоматически, а при необходимости их можно изменить вручную.

## Как задать значения по умолчанию

1. Внизу боковой панели нажмите **Настройки**.
2. В разделе **Настройки организации** нажмите **Настройки**.
3. Откройте пункт **Значения по умолчанию**.
4. Нажмите на иконку **карандаша** справа от нужного значения.
5. Укажите нужное значение.
6. Нажмите **синюю галочку**, чтобы подтвердить действие,  
   или **красный крестик**, чтобы отменить.

---

## Какие значения можно задать по умолчанию

- **Валюта** — валюта, которую вы чаще используете в документах.
- **Склад** — основной склад для закупок и отгрузок.
- **Касса** — касса для приема оплат от клиентов.
- **Цена продажи** — прайс-лист для основной части продаж.
- **Цена межорг. отгрузок** — прайс-лист для отгрузок другим организациям.
- **Цена закупки** — прайс-лист, который чаще используется в закупках.
- **Единица хранения** — основная единица измерения товаров.
- **Резервирование заказа** — резервировать ли товар при создании заказа.
- **Вид акта сверки** — формат отображения акта сверки  
  (по документам, товарам или датам).
- **Режим скидки** — указывать скидку в сумме или в процентах.
- **Режим направления скидки** — использовать скидки или наценки.
- **Дата в списке документов** — фильтр по дате, применяемый автоматически.
- **Статусы документов** — статус, с которым создаются новые документы.
- **Период консигнации** — срок оплаты после отгрузки в долг.
- **Время на логистику** — среднее время доставки товара на склад.
- **Префикс телефонов** — код страны для контактов контрагентов.
- **Минимальная и максимальная сумма заказа** — ограничение суммы заказа.
- **Карта** — карта для просмотра адресов (Yandex, Google, 2GIS).
- **Тип штрихкода** — тип штрихкода при создании товаров.
