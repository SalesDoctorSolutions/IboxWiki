# Использовать мультивалютную оплату

:::info Информация
В iBox взаиморасчеты с контрагентами формируются **отдельно по каждой валюте**.  
Для погашения долга в одной валюте необходимо производить оплату **в этой же валюте**.
:::

Данная функция позволяет **продавать товар в одной валюте, а принимать оплату в другой**.

После включения функционала в документах оплаты можно отдельно указывать:

- **Валюту оплаты** — валюта, в которой деньги поступают или списываются из кассы;
- **Валюту документа** — валюта, в которой формируются взаиморасчеты с контрагентом;
- **Курс валюты** для конкретного документа.

---

## Когда полезно использовать мультивалютную оплату

Использование функции помогает:

- принимать оплату от клиентов в удобной для них валюте;
- работать с иностранными поставщиками и покупателями;
- корректно учитывать взаиморасчеты при конвертации валют;
- уменьшать количество дополнительных операций по обмену валют;
- вести более точный финансовый учет при работе с несколькими валютами.

Это особенно полезно, если:

- компания продает товары в одной валюте, а фактические оплаты получает в другой;
- используются несколько касс с разными валютами;
- бизнес работает с международными контрагентами;
- часто меняется курс валют.

---

## Как включить функцию

1. Перейдите в **Настройки** на левой панели.
2. В настройках организации откройте раздел **Настройки**.
3. Перейдите в раздел **Функционал**.
4. Найдите параметр **«Использовать мультивалютную оплату»** и нажмите на иконку **карандаша** справа.
5. Выберите опцию **Да**.
6. Нажмите на **синюю иконку галочки**, чтобы сохранить изменения.
