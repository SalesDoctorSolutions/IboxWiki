## Заказы

Отчет **Заказы** содержит подробный список всех заказов с разбивкой по каждому товару.

Используйте отчет, чтобы:

- Просматривать детали по каждому заказу
- Анализировать заказы по товарам
- Контролировать цены, скидки и итоговые суммы

## Как открыть отчет

1. Откройте раздел **Отчеты** в левой панели.
2. В блоке **Отчеты по продажам** выберите **Заказы**.

Откроется страница отчета с данными за выбранный период.

## Доступные поля

В отчете можно отобразить следующие поля:

- **Товар** — наименование товара в заказе  
- **Дата документа** — дата заказа  
- **Номер документа** — номер заказа в системе  
- **Клиент** — покупатель, оформивший заказ  
- **Ответственный** — сотрудник, закрепленный за заказом  
- **Количество** — количество товара в заказе  
- **Цена** — цена за единицу товара  
- **Сумма** — итоговая сумма по позиции  
- **Артикул** — внутренний или производственный код товара  
- **Посредник** — партнер или агент, участвующий в сделке  
- **Скидка** — размер скидки по позиции  
- **Сумма скидок** — общая сумма предоставленной скидки  
- **Цена до скидки** — цена товара без учета скидки  
- **Сумма до скидки** — сумма по позиции без учета скидки  
- **Категория** — категория товара  
- **Бренд** — торговая марка товара  
- **Прайс-лист** — прайс-лист, примененный к заказу
