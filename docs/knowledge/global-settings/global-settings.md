# Общие настройки

## Что такое общие настройки
Общие настройки — это параметры, которые применяются ко всем организациям в системе.  

---

## Где настроить общие настройки
Общие настройки можно задать только через организацию, которая является **головным офисом**:

1. Перейдите в **Настройки**.
2. Откройте раздел **Общие настройки**.

---

## Что можно задать через общие настройки
Через общие настройки можно настроить:

- валюты и курсы;
- единицы измерения;
- способы оплаты;
- категории клиентов;
- роли сотрудников;
- категории товаров;
- группы товаров;
- бренды.
