# Настройка клиентов и поставщиков в организациях

## Как добавить клиентов и поставщиков в организации
1. Перейдите в **панель администратора**.
2. Откройте раздел **Контрагенты → Контрагенты**.
3. Выберите клиентов и поставщиков, которых хотите добавить.
4. Нажмите на **три точки** на верхней панели.
5. Выберите **Добавить в организацию**.
6. Отметьте нужные организации.
7. Нажмите **Применить**.

---

## Как убрать клиентов и поставщиков из организаций
1. Перейдите в **панель администратора**.
2. Откройте раздел **Контрагенты → Контрагенты**.
3. Выберите клиентов и поставщиков, которых хотите убрать.
4. Нажмите на **три точки** на верхней панели.
5. Выберите **Удалить из организации**.
6. Отметьте организации, из которых нужно убрать контрагентов.
7. Нажмите **Применить**.

---

## Как создавать клиентов и поставщиков сразу во всех организациях

Если вы хотите, чтобы при создании клиента или поставщика в одной организации он автоматически появлялся во всех остальных:

1. Перейдите в **панель администратора**.
2. Откройте раздел **Настройки → Глобальные настройки**.
3. Поставьте галочку **«Автоматически прикреплять новых клиентов/поставщиков ко всем организациям»**.
4. Нажмите **Сохранить**.

Если вы не хотите автоматическое добавление — снимите эту галочку.

---

## Как установить общий лимит задолженности для клиентов по всем организациям

Если вы хотите ограничить продажу в долг клиенту, который задолжал одной из организаций, во всех остальных организациях, установите кредитный лимит на уровне всей системы.

1. Перейдите в **панель администратора**.
2. Откройте раздел **Настройки → Глобальные настройки**.
3. Поставьте галочку **«Применять кредитный лимит на уровне всей системы»**.
4. Нажмите **Сохранить**.

После этого клиент не сможет покупать товары в долг, если его суммарная задолженность по всем организациям превышает установленный лимит.

:::info **Информация**  
Кредитный лимит для клиента можно задать при его создании или редактировании.
:::
