# Движения по кассе

Отчет **Движения по кассе** показывает список всех денежных операций — все приходы и расходы по кассам.

Используйте отчет, чтобы:

- Просматривать историю всех денежных операций  
- Контролировать приходы и расходы по каждой кассе  
- Анализировать движение денежных средств  

## Как открыть отчет

1. Откройте раздел **Отчеты** в левой панели.
2. В блоке **Отчеты по финансам** выберите **Движения по кассе**.

Откроется страница отчета с данными за выбранный период.

## Доступные поля

В отчете можно отобразить следующие поля:

- **Дата** — дата денежной операции  
- **Источник** — источник операции (например: оплата клиента, оплата поставщику, возвраты средств)  
- **Приход** — сумма поступления в кассу  
- **Расход** — сумма списания из кассы  
- **Касса** — касса, по которой проведена операция  
- **Ответственный** — сотрудник, ответственный за операцию  
- **Создал** — пользователь, создавший запись
