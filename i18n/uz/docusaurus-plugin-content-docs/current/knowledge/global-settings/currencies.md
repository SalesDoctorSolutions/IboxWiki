# Valyutalar 

## Valyuta qo‘shish

Yangi valyuta qo‘shish uchun:

1. Chap panelda **Sozlamalar** bo‘limiga bosing.  
2. Pastga tushib, **Umumiy sozlamalar** ichidan **Valyutalar** bo‘limiga o‘ting.  
3. O‘ng yuqori burchakda **+ Valyuta** tugmasini bosing.  
4. Quyidagi ma’lumotlarni kiriting:
   - valyuta kodi;
   - belgi;
   - kurs.
5. **Saqlash** tugmasini bosing.

---

## Asosiy valyutani tanlash

1. Chap panelda **Sozlamalar** bo‘limiga bosing.  
2. **Umumiy sozlamalar → Valyutalar** bo‘limiga o‘ting.  
3. Joriy asosiy valyuta ustiga kursorni olib boring.  
4. **Asosiy valyutani o‘zgartirish** tugmasini bosing.  
5. Kerakli valyutani tanlang.  
6. **Saqlash** tugmasini bosing.

:::warning **Muhim**  
Asosiy valyutani faqat tizimda hali hujjatlar yaratilmagan paytda tanlash mumkin.  
Agar hujjatlar mavjud bo‘lsa, asosiy valyutani o‘zgartirib bo‘lmaydi.
:::

---

## Valyuta kursini o‘zgartirish

1. **Sozlamalar → Umumiy sozlamalar → Valyutalar** bo‘limiga o‘ting.  
2. Kerakli valyuta ustiga kursorni olib boring.  
3. ✏️ **Qalamcha** (tahrirlash) belgisini bosing.  
4. Yangi kursni kiriting.  
5. **Saqlash** tugmasini bosing.

:::info **Ma’lumot**  
Kurs o‘zgartirilganda, oldingi hujjatlardagi kurs qiymatlari o‘zgarishsiz qoladi.
:::

---

## Kurs tarixini ko‘rish

1. **Sozlamalar → Umumiy sozlamalar → Valyutalar** bo‘limiga o‘ting.  
2. Kerakli valyuta ustiga kursorni olib boring.  
3. 📅 **Kalendar** belgisini bosing.
