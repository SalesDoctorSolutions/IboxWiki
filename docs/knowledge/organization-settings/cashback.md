# Кешбек

## Как работает кешбек-система в iBOX
Кешбек в iBOX работает как накопительная система. После того как совершается отгрузка клиенту, ему начисляется кешбек в отдельный кешбек-баланс. Кешбек можно применить только во время оформления отгрузки.

---

## Как включить программу кешбека
1. Нажмите **Настройки** в нижней части боковой панели.
2. В настройках организации перейдите в раздел **Кешбек**.
3. Поставьте галочку **Включить кешбек**.
4. Укажите процент кешбека по умолчанию.
5. Нажмите **Сохранить**.

---

## Как начислять и использовать кешбек

### Начисление кешбека
Во время создания отгрузки под списком товаров есть опция **Начислить кешбек**.  
Она включена по умолчанию, если кешбек активирован в системе.

Если вы не хотите начислять кешбек за данную отгрузку, просто отключите эту опцию.

:::warning **Важно:**  
Клиент сможет использовать начисленный кешбек только после оплаты долга.
:::

---

### Использование кешбека
Чтобы использовать кешбек во время отгрузки:

1. Нажмите **Использовать кешбек** в правой части экрана под списком товаров.
2. Введите сумму кешбека, которую хотите применить.
3. Нажмите **Сохранить**.

---

## Приоритет кешбека
Вы можете установить отдельные проценты кешбека:
- для конкретных товаров;
- для конкретных клиентов;
- по системе (общий процент).

Проценты кешбека **не суммируются**. Работает следующий приоритет:

1. Кешбек для товара.
2. Кешбек для клиента.
3. Кешбек по системе.

**Пример:**  
- По системе — 1%  
- Для клиента — 5%  
- Для товара — 3%  

В этом случае будет применён кешбек **3%** (по товару).  
Если кешбек для товара убрать, применится **5%** (для клиента).  
Кешбек по системе (1%) сработает только если нет других настроек.
