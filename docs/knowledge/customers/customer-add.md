## Добавление клиентов

В iBox вы можете добавить клиентов вручную, импортировать их из Excel или прикрепить из другой организации.

---

## Как добавить клиента вручную

1. Перейдите в раздел **Клиенты**.
2. В правом верхнем углу нажмите на **плюс**.
3. В открывшейся форме:
   - заполните **название**
   - укажите **категорию клиента**,
   - заполните **основной номер телефона** и **Telegram-номер**,
   - при необходимости добавьте **фото**.
4. Если есть дополнительные номера, откройте вкладку **Дополнительные контакты** и заполните ее.
5. Добавьте **адрес клиента**:
   - введите вручную,
   - или выберите локацию на карте.
6. Заполните **реквизиты клиента**.
7. Перейдите во вкладку **Прочее** и укажите:
   - прайс-лист для клиента,
   - кредитный лимит,
   - период консигнации,
   - периодичность отгрузок,
   - код,
   - сортировочный номер,
   - комментарий.
8. Нажмите **Сохранить**.

## Как добавить клиентов из другой организации

Иногда пользователи создают нового клиента во второй организации, даже если он уже существует в системе. Делать так не рекомендуется.

### Почему не стоит создавать клиента заново
- Возникают дубликаты.
- Появляются ошибки в расчетах.
- Теряется связь с клиентом из другой организации.
- Один Telegram-номер нельзя привязать к нескольким клиентам (он уникален).

### Как правильно добавить клиента из другой организации

1. Перейдите в раздел **Клиенты**.
2. В правом верхнем углу нажмите на **три точки**.
3. Выберите **Добавить из других организаций**.
4. В окне выберите нужных клиентов.
5. Нажмите **Применить**.

## Как загрузить клиентов из Excel

Если у вас есть готовый список клиентов, вы можете импортировать его в систему.

### Обязательные поля
- Название клиента

### Дополнительные поля
- Основной номер телефона
- Telegram-номер
- Категория
- Начальный баланс (по каждой валюте отдельно)
- Кредитный лимит
- Адрес
- ИНН
- Примечание

### Как выполнить импорт

1. Перейдите в раздел **Клиенты**.
2. Нажмите на **три точки** в правом верхнем углу.
3. Выберите **Импорт с Excel**.
4. Перетащите файл в окно импорта или выберите его на компьютере.
5. В пункте **Что делать с дубликатами?** выберите  
   **Добавить дубликаты как новых контрагентов**.
6. Сопоставьте столбцы файла с полями системы.
7. Нажмите **Начать импорт**.
