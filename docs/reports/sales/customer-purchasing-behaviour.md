# Покупательское поведение клиентов

Отчет **Покупательское поведение клиентов** отображает динамику работы с клиентами по месяцам.

Отчет представлен в виде таблицы:

- Каждая строка — отдельный клиент  
- Каждый столбец — месяц периода  

Отчет помогает анализировать торговые отношения с клиентами и выявлять изменения в объемах продаж.

## Как открыть отчет

1. Откройте раздел **Отчеты** в левой панели.
2. В блоке **Отчеты по продажам** выберите **Покупательское поведение клиентов**.

## Как интерпретировать данные

Все значения в отчете отображаются в процентах.

- **100%** — месяц с максимальным объемом продаж по клиенту (пик активности);  
- **0%** — месяц с минимальным объемом продаж;  
- остальные значения показывают относительный уровень продаж между минимумом и максимумом.

Проще говоря, система сравнивает каждый месяц с худшим и лучшим месяцем клиента и показывает, насколько текущий месяц близок к максимуму.

:::info Информация
Если значение **0%** сохраняется несколько месяцев подряд, это может указывать на снижение активности или потерю клиента.
:::

## Учет возвратов

При формировании отчета вы можете выбрать способ расчета:

- По общей сумме отгрузок  
- С учетом возвратов  

Чтобы учитывать возвраты в расчете, установите флажок **Учитывая возвраты**.
