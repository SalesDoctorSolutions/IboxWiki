# Удаление и деактивация поставщиков

В iBox есть несколько способов убрать поставщика из списков: удалить его, деактивировать или добавить в чёрный список.

### Как удалить поставщика
Удаление используется только в том случае, если поставщик был создан по ошибке.  
Если с поставщиком уже есть история, система не позволит его удалить. Очистка истории может привести к ошибкам в отчётах.

Чтобы удалить поставщика:
1. Перейдите в раздел **Поставщики**.
2. Наведите курсор на правую часть строки нужного поставщика.
3. Нажмите на иконку 🗑 **корзины**.
4. Подтвердите удаление.

---

### Как деактивировать поставщика
Если с поставщиком есть история, но вы больше не работаете с ним, его можно деактивировать.  
Поставщик исчезнет из рабочих списков, но сохранится в отчётах.

Чтобы деактивировать поставщика:
1. Перейдите в раздел **Поставщики**.
2. Выберите одного или несколько поставщиков.
3. Нажмите на **три точки** в верхней части экрана.
4. Выберите опцию **Сделать неактивным**.

:::warning **Важно**  
Если нужно убрать поставщика только из одной организации, но продолжать использовать его в других, настройте организации поставщика через панель администратора.
:::

---

### Чёрный список
Если вы не хотите оформлять закупки у поставщика, но вам ещё нужно вносить ему оплату, добавьте его в чёрный список:

1. Перейдите в раздел **Поставщики**.
2. Наведите курсор на правую часть строки выбранного поставщика.
3. Нажмите на иконку ✏️ **карандаша** (редактировать).
4. Откройте вкладку **Прочее**.
5. Поставьте галочку **В чёрном списке**.

---

### Что выбрать

| Ситуация | Действие |
|----------|----------|
| Поставщик создан по ошибке | Удаление поставщика |
| Не работаете с поставщиком в одной организации, но используете в других | Настройка организаций поставщика |
| Больше не работаете с поставщиком, но у него есть история | Деактивация поставщика |
| Не хотите оформлять закупки у поставщика, но хотите вносить ему оплату | Чёрный список |
