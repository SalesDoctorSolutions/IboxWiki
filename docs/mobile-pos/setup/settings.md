# Настройки и конфигурации

При первом входе в мобильное приложение iBox открывается экран **Настроек**.  
Позже вы можете вернуться к нему через **меню (иконка ☰) → Настройки**.

В этом разделе вы можете:

- изменить язык интерфейса;
- установить локальный пароль;
- подключить термопринтер;
- задать значения по умолчанию;
- настроить поведение приложения.

---

## Изменение языка системы

1. Откройте **меню (☰) → Настройки**.  
2. Нажмите **Язык**.  
3. Выберите нужный язык.

Изменения применяются сразу.

---

## Локальный пароль (PIN-код)

Вы можете защитить приложение с помощью PIN-кода.

### Как установить пароль

1. Перейдите в **меню (☰) → Настройки**.  
2. Откройте раздел **Код-пароль**.  
3. Введите PIN-код (4 цифры).  
4. Подтвердите пароль.

После установки вы можете дополнительно включить:

- **Face ID** (для iOS);  
- **отпечаток пальца** (для Android).

### Как отключить пароль

1. Перейдите в **Настройки → Код-пароль**.  
2. Отключите пароль.

---

## Значения по умолчанию

Вы можете настроить значения по умолчанию, чтобы ускорить работу и не вводить данные вручную каждый раз.

Для этого:

1. Откройте **Настройки**.  
2. Нажмите на нужный параметр.  
3. Выберите значение.

### Доступные настройки

- **Язык накладного** — язык печатных документов;  
- **Клиент** — автоматически подставляется в продажах;  
- **Поставщик** — используется в закупках;  
- **Касса** — применяется в финансовых операциях;  
- **Склад** — основной склад для операций;  
- **Прайс-лист** — используется при продажах и возвратах;  
- **Валюта по умолчанию** — применяется в денежных операциях;  
- **Шаблон печати** — печать с ценами или без цен;  
- **Тип скидки**:
  - скидка в сумме;  
  - скидка в процентах;  
  - наценка в сумме;  
  - наценка в процентах.  

---

:::tip Совет

- Настройте значения по умолчанию перед началом работы — это ускорит создание документов.  
- Установите PIN-код для защиты данных.  
- Проверьте настройки печати, если используете накладные или чеки.
:::
