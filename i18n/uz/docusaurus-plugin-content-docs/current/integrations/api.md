# iBox API va integratsiyalar

iBox tashqi tizimlar: saytlar, CRM, mobil ilovalar va boshqa servislar bilan integratsiya qilish uchun API taqdim etadi.

API yordamida siz quyidagilarni qilishingiz mumkin:

- mahsulotlar, mijozlar va hujjatlar bo‘yicha ma’lumotlarni olish;
- buyurtmalar, sotuvlar va boshqa operatsiyalarni yaratish;
- biznes jarayonlarini avtomatlashtirish;
- iBox’ni ichki tizimlaringiz bilan integratsiya qilish.

---

## API kalitini qanday olish mumkin

API kaliti API bilan ishlashda avtorizatsiya uchun ishlatiladi.

Kalitni faqat administrator huquqiga ega foydalanuvchi yaratishi mumkin.

### Qadamlar

1. Yuqori paneldagi **korxona nomi** ustiga bosing.  
2. **Boshqaruv** bo‘limini tanlang.  
3. Chap panelda **Sozlamalar** bo‘limiga o‘ting.  
4. **API kalitlari** bo‘limini oching.  
5. **Plus** tugmasini bosing.  
6. Quyidagilarni to‘ldiring:
   - **Kalit nomi** — qulay aniqlash uchun;  
   - **Korxonalar** — qaysi korxonalarga kirish bo‘lishini tanlang.  
7. **Saqlash** tugmasini bosing.  
8. Yaratilgan kalitni nusxalab oling.

:::warning Muhim
API kalitini yaratilgan zahoti saqlab qo‘ying.

Oyna yopilgandan keyin siz uni **qayta ko‘ra olmaysiz**.  
Agar kalit yo‘qolsa — yangisini yaratish kerak bo‘ladi.
:::

---

## API kalitlarini boshqarish

### Kalitni qanday o‘chirish mumkin

Agar kalit endi ishlatilmasa yoki kirishni cheklash kerak bo‘lsa:

1. **Boshqarish → Sozlamalar → API kalitlari** bo‘limini oching.  
2. Kerakli kalitni toping.  
3. O‘ng tomondagi **⋮ (uch nuqta)** menyusini bosing.  
4. **Faolsizlantirish** bandini tanlang.

Faolsizlantirgandan keyin kalit ishlamaydi.

---

## Xavfsizlik bo‘yicha tavsiyalar

- API kalitini uchinchi shaxslarga bermang;
- kalitlarni xavfsiz joyda saqlang (masalan, muhit o‘zgaruvchilarida);
- turli integratsiyalar uchun alohida kalitlardan foydalaning;
- agar kalit oshkor bo‘lganidan shubhalansangiz, uni darhol faolsizlantirib, yangisini yarating.
