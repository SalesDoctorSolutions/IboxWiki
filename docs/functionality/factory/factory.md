Модуль **Фабрика** позволяет производить товары из сырья, рассчитывать себестоимость и управлять производственными операциями.

Модуль подходит для компаний, которые занимаются производством — например, в сфере стройматериалов, мебели, продуктов питания или сборки комплектов.

## Когда полезно использовать модуль

Модуль особенно полезен, если:

- вы производите товары из сырья или комплектующих;
- нужно автоматически списывать материалы и приходовать готовую продукцию;
- важно точно рассчитывать себестоимость продукции;
- используются повторяющиеся производственные процессы;
- требуется учитывать производство внутри склада без сторонних систем;
- вы собираете комплекты товаров для продажи.

## Как включить модуль

1. Откройте **Настройки** на левой панели.  
2. Перейдите в раздел **Настройки организации**.  
3. Откройте вкладку **Функционал**.  
4. Найдите параметр **Использовать сборки** и нажмите на иконку **карандаша**.  
5. Выберите значение **Да**.  
6. Нажмите на **галочку**, чтобы сохранить изменения.

После включения в меню появится раздел **Фабрика** со следующими подразделами:

- **Сборки**  
- **Техкарты**  

## Основные понятия

### Сборка

**Сборка** — это операция, при которой сырье списывается со склада и превращается в готовую продукцию.

В сборке указываются:

- склад списания сырья;
- склад поступления готовой продукции;

**Пример:**

- производство бетонной смеси из цемента, песка и щебня;
- сборка мебельного комплекта из отдельных деталей;
- формирование набора товаров для продажи.

### Техкарта

**Техкарта** — это шаблон сборки. В ней заранее задан состав и пропорции сырья.

В техкарте указываются:

- сырье и комплектующие;
- количество каждого компонента;
- выход готовой продукции.

Техкарты удобно использовать для регулярного производства.

**Пример:**

- рецепт бетонной смеси с фиксированными пропорциями;
- стандартная комплектация шкафа;
- постоянный состав строительного набора.

### Разница между сборкой вручную и по техкарте

| Параметр | Ручная сборка | Сборка по техкарте |
|----------|----------------|-------------------|
| **Создание** | Заполняется вручную каждый раз | Создается на основе техкарты |
| **Когда использовать** | Разовые операции | Регулярное производство |
| **Гибкость** | Любые составы и количества | Фиксированные нормы |
| **Пример** | Набор стройматериалов под заказ | Стандартная бетонная смесь |

## Расчет себестоимости

Система автоматически рассчитывает себестоимость готовой продукции на основе себестоимости сырья.

По умолчанию себестоимость распределяется равномерно между всеми произведенными товарами.

**Если нужно изменить расчет:**

- укажите долю себестоимости для каждого товара в процентах;
- система распределит затраты с учетом указанных долей.

**После проведения сборки вы можете посмотреть:**

- итоговую себестоимость готовой продукции;
- детализацию расчета по каждому компоненту.
