# Отгрузки

Отчет **Отгрузки** содержит подробный список всех отгрузок и возвратов с разбивкой по каждому товару.

:::info Информация
Чтобы просмотреть только возвраты, настройте фильтр **Тип документа**.  
Возвраты в отчете подсвечиваются красным цветом.
:::

Используйте отчет, чтобы:

- Контролировать объем отгруженных товаров
- Анализировать выручку по отгрузкам
- Проверять примененные цены и скидки

## Как открыть отчет

1. Откройте раздел **Отчеты** в левой панели.
2. В блоке **Отчеты по продажам** выберите **Отгрузки**.

Откроется страница отчета с данными за выбранный период.

## Доступные поля

В отчете можно отобразить следующие поля:

- **Товар** — наименование отгруженного товара  
- **Дата документа** — дата создания документа  
- **Документ К-во / Номер** — номер документа или количество документов при группировке  
- **Клиент К-во / Название** — наименование клиента или количество клиентов при группировке  
- **Ответственный** — сотрудник, закрепленный за документом  
- **Количество** — количество товара в документе  
- **Цена** — цена за единицу товара  
- **Сумма** — итоговая сумма по позиции  
- **Артикул** — код товара  
- **Тип документа** — отгрузка или возврат  
- **Посредник** — партнер или агент
- **Скидка в процентах** — размер скидки в процентах  
- **Скидка** — сумма скидки по позиции  
- **Сумма скидок** — общая сумма скидок по позиции  
- **Цена до скидки** — цена без учета скидки  
- **Сумма до скидки** — сумма без учета скидки  
- **Категория** — категория товара  
- **Бренд** — торговая марка товара  
- **Прайс-лист** — прайс-лист, примененный к документу
