# Boshqa korxonalarga to‘lovlar

## Boshqa korxonaga to‘lov jo‘natish

1. **Korxonalar → Korxonalaro to‘lovlar** bo‘limiga o‘ting  
2. **To‘lov jo'natishlar** yorlig‘ini oching  
3. O‘ng yuqori burchakdagi **+ To‘lov jo‘natish** tugmasini bosing  
4. Qabul qiluvchi korxonani tanlang  
5. Ma’lumotlarni kiriting:  
   - summa  
   - valyuta  
   - kassa  
     (zarur bo‘lsa qo‘shimcha kassalar qo‘shing)  
6. **Saqlash** tugmasini bosing  

---

## Boshqa korxonadan kelgan to‘lovni qabul qilish

1. **Korxonalar → Korxonalararo to‘lovlar** bo‘limiga o‘ting  
2. **To‘lov qabul qilishlar** yorlig‘ini oching  
3. Kerakli hujjat raqamini bosing  
4. Ekran pastidagi statusni bosing  
5. Kerakli statusni tanlang:  
   - **Qabul qilingan** — agar to‘lovni qabul qilmoqchi bo‘lsangiz  
   - **Rad etilgan** — agar to‘lovni rad etmoqchi bo‘lsangiz yoki xatoni tuzatish kerak bo‘lsa  
6. Pul tushadigan kassani tanlang  

:::info Ma’lumot  
To‘lovni bir nechta kassaga qabul qilishingiz mumkin.  

Buning uchun **+** belgisini bosing, kassalarni tanlang va summalarni kiriting.  
:::
