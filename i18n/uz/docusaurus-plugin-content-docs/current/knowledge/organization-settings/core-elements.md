# Asosiy elementlarni qo‘shish

## Kassalarni qo‘shish

Kassalarni qo‘shish turli to‘lov usullari bo‘yicha balanslarni ajratib yuritish uchun kerak  
(masalan, naqd pul kassasi, terminal yoki o‘tkazma), xodimlar yoki boshqa mezonlar bo‘yicha.

Kassa qo‘shish uchun:

1. Yon panelning pastki qismidan **Sozlamalar**ni bosing.  
2. Sozlamalarda **Kassalar** bo‘limiga o‘ting.  
3. O‘ng yuqori burchakda **+ Kassa** tugmasini bosing.  
4. Quyidagilarni kiriting:
   - nomi;  
   - to‘lov usuli;  
   - tartiblash soni.  
5. **Saqlash** tugmasini bosing.

---

## Omborlarni qo‘shish

Ombor qo‘shish mahsulot qoldiqlarini turli joylar bo‘yicha,  
mahsulot turlari bo‘yicha (masalan, xomashyo va tayyor mahsulot), xodimlar yoki boshqa mezonlar asosida ajratib yuritish uchun kerak.

Ombor qo‘shish uchun:

1. Yon panelning pastki qismidan **Sozlamalar**ni bosing.  
2. Sozlamalarda **Omborlar** bo‘limiga o‘ting.  
3. O‘ng yuqori burchakda **+ Ombor** tugmasini bosing.  
4. Quyidagilarni kiriting:
   - nomi;  
   - manzil;  
   - tartiblash soni.  
5. **Saqlash** tugmasini bosing.
