# Korxonalaro buyurtmalar

Korxonalar o‘rtasidagi buyurtmalar ta'minotchiga beriladigan buyurtmalar kabi ishlaydi.

- Birinchi korxona ikkinchi korxonaga buyurtma yuboradi — xuddi ta'minotchiga buyurtma bergandek  
- Ikkinchi korxona bu buyurtmani mijoz buyurtmasi sifatida qabul qiladi va birinchi korxonaga sotuv rasmiylashtiradi  

---

## Boshqa korxonaga buyurtma yuborish

Agar mahsulot yetishmasa, uni boshqa korxonadan buyurtma qilishingiz mumkin:

1. **Korxonalar → Korxonalaro buyurtmalar** bo‘limiga o‘ting  
2. **Bizning buyurtmalar** yorlig‘ini oching  
3. **+ Buyurtma jo'natish** tugmasini bosing  
4. Ochilgan shaklda korxonani tanlang  
5. Hujjat valyutasini tanlang  
6. Mahsulotlarni qo‘shing  
7. Mahsulot ma’lumotlarini kiriting:  
   - miqdor  
   - narxlar  
8. **Saqlash** tugmasini bosing  

---

## Boshqa korxonadan kelgan buyurtmani qabul qilish

1. **Korxonalar → Korxonalaro buyurtmalar** bo‘limiga o‘ting  
2. **Bizlarga buyurtmalar** yorlig‘ini oching  
3. Kerakli hujjat raqamini bosing  
4. Ekran pastidagi hujjat statusini bosing  
5. Kerakli statusni tanlang:  
   - **Qabul qilingan** — agar buyurtmani qabul qilmoqchi bo‘lsangiz  
   - **Rad etilgan** — agar buyurtmani rad etmoqchi bo‘lsangiz  

---

## Buyurtma asosida jo'natma rasmiylashtirish

Buyurtma qabul qilingandan so‘ng mahsulot jo‘natmasini rasmiylashtirishingiz mumkin:

1. **Korxonalar → Korxonalaro buyurtmalar** bo‘limiga o‘ting  
2. **Bizlarga buyurtmalar** yorlig‘ini oching  
3. Kerakli hujjat raqamini bosing  
4. O‘ng yuqori burchakdagi **Jo'natma yaratish** tugmasini bosing  
5. Ma’lumotlarni tekshiring va kerak bo‘lsa o‘zgartiring  
6. **Saqlash** tugmasini bosing
