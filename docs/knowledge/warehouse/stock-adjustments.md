# Корректировки

Корректировка помогает привести данные в порядок: добавить недостающие позиции, убрать лишние, исправить количество.

---

## Когда использовать корректировку
- вы вводите начальные остатки в систему;
- вы добавляете свой товар в систему;
- реальные остатки на складе не совпадают с данными в системе;
- списание бракованных или потерянных товаров;
- расход товара в личных целях.

:::warning **Важно!**  
Не используйте корректировку для прихода товара.  
Для этого используйте **закупку**.  
При закупке формируются взаиморасчёты с поставщиком, при корректировке — нет.
:::

---

## Как оформить корректировку
1. Перейдите в раздел **Корректировки** в модуле **Склад**.
2. Нажмите на **плюс** в правом верхнем углу.
3. Выберите валюту документа.
4. Выберите склад, в который вы хотите внести корректировку.
5. Добавьте товары:
   - через подбор товаров;
   - или загрузите список из Excel.
6. Введите количество  
   (если нужно уменьшить количество, используйте знак минуса, например: **-5**).
7. Заполните приходную цену для товаров с положительной корректировкой.
8. Нажмите **Сохранить**.

:::info **Информация:**  
Себестоимость товаров с отрицательной корректировкой рассчитывается по принципу **First In First Out (FIFO)**.
:::
