# Ta'minotchiga qaytarishlar

Qaytarish mahsulot ta'minotchiga to‘liq yoki qisman qaytarilganda rasmiylashtiriladi — masalan, nuqsonli bo‘lsa, noto‘g‘ri tur kelib qolsa yoki xarid bekor qilinganda.

---

### Xarid asosida ta'minotchiga qaytarish rasmiylashtirish

Siz allaqachon amalga oshirilgan xarid bo‘yicha ta'minotchiga qaytarish yaratishingiz mumkin.  
Bu holatda mahsulotlar ombordan aynan o‘sha xariddagi tannarx bo‘yicha chiqariladi.

1. **Xaridlar** bo‘limiga o‘ting.  
2. Ta'minotchiga qaytarish qilmoqchi bo‘lgan xarid raqamiga bosing.  
3. Yuqori o‘ng burchakda **Mahsulot qaytarish** tugmasini bosing.  
4. Qaytariladigan mahsulot miqdorini kiriting.  
5. **Saqlash** tugmasini bosing.

---

### Xaridga bog‘lanmagan ta'minotchiga qaytarish

Xaridga bog‘lamasdan ham ta'minotchiga qaytarish rasmiylashtirish mumkin.  
Bu holatda ombordan chiqariladigan mahsulot tannarxi **FIFO (First In First Out — birinchi kirgan birinchi chiqadi)** tamoyiliga asosan hisoblanadi.

#### Bog‘lanmagan ta'minotchiga qaytarishni qanday rasmiylashtirish

1. **Xaridlar → Ta'minotchiga qaytarishlar** bo‘limiga o‘ting.  
2. Yuqori o‘ng burchakdagi ➕ tugmasini bosing.  
3. Qaytarish qilinadigan ta'minotchini tanlang.  
4. Valyutani belgilang.  
5. Mahsulotlar qaytariladigan omborni tanlang.  
6. Mahsulotlarni qo‘shing.  
7. Mahsulot ma’lumotlarini kiriting:
   - miqdor  
   - narxlar (prays-listni tanlang yoki qo‘lda kiriting)  
8. **Saqlash** tugmasini bosing.

---

### Ta'minotchidan pulni qaytarib olish

Ta'minotchi qaytarish rasmiylashtirilgach, ta'minotchi mahsulotlar uchun pulni qaytarishi kerak.
1. **Ta'minotchiga qaytarishlar** bo‘limiga o‘ting.  
2. Kerakli hujjat raqamiga bosing.  
3. Yuqori o‘ng burchakda **Pul qaytarish** tugmasini bosing.  
4. Ma’lumotlarni tekshiring va kerak bo‘lsa o‘zgartiring.  
5. **Saqlash** tugmasini bosing.
