# Разделение отгрузок по складам

Данная функция позволяет **автоматически разделять документ отгрузки по каждому складу**.

Если при создании отгрузки в списке товаров указаны **разные склады**, то при сохранении система создаст **несколько отдельных документов** — по одному для каждого склада со своим списком товаров.

---

:::info Информация
Данная функция работает **только при создании нового документа** и не применяется при редактировании уже созданной отгрузки.
:::

---

## Когда полезно использовать функцию

Использование функции помогает:

- автоматически распределять отгрузки по складам без ручного разделения;
- упрощать учет остатков на каждом складе;
- уменьшать количество ошибок при оформлении отгрузок;
- ускорять работу сотрудников при оформлении документов;
- корректно контролировать процесс сборки и отправки товаров с разных складов.

Это особенно полезно, если:

- компания использует несколько складов;
- товары в одной продаже отгружаются из разных складов;
- необходимо четко контролировать движение товаров по каждому складу;
- складские операции выполняют разные сотрудники.

---

## Как включить функцию

1. Перейдите в **Настройки** на левой панели.
2. В настройках организации откройте раздел **Настройки**.
3. Перейдите в раздел **Функционал**.
4. Найдите параметр **«Разделение отгрузок по складам»** и нажмите на иконку **карандаша** справа.
5. Выберите опцию **Да**.
6. Нажмите на **синюю иконку галочки**, чтобы сохранить изменения.
