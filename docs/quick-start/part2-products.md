# Часть 2. Товары, цены и остатки

В этом разделе вы узнаете, как:
- Добавлять товары 
- Создавать прайс-листы  
- Вносить начальные остатки на складах  

---

## Единицы измерения

Сначала добавьте единицы измерения, в которых вы будете учитывать и продавать товары.

1. Перейдите в **Настройки**.
2. В **Общих настройках** откройте раздел **Единицы измерения**.
3. Нажмите на плюс в правом верхнем углу.
4. Укажите:
    - Название единицы  
    - Короткое название (до 3 символов)
<img src="/img/quick-start/qs-uom.png" alt="Добавление единиц измерения в iBox" width="600" />
5. Нажмите **Сохранить**.

---

## Категории товаров

Категории нужны для удобной навигации и анализа отчетов по группам товаров.

1. В **Общих настройках** откройте раздел **Категории товаров**.
2. Нажмите на плюс в правом верхнем углу.
3. Укажите название категории.
4. Задайте сортировочный номер — чем больше значение, тем выше категория отображается в списке.
<img src="/img/quick-start/qs-category-add.png" alt="Создание категорий в iBox" width="600" />
5. Нажмите **Сохранить**.

:::tip **Совет**  
Для более детальных отчетов добавьте бренды и группы товаров в общих настройках.
:::

## Добавление товаров 

### Загрузка товаров из Excel

Если у вас уже есть список товаров, вы можете загрузить его в систему.

1. Подготовьте файл **Excel**.

**Обязательные поля:**
    - Название товара  
    - Единица хранения  

**Дополнительные поля:**
    - Категория товара  
    - Начальные остатки (отдельно по каждому складу)  
    - Приходная цена начальных остатков (в базовой валюте)

:::tip Совет  
Указывайте реальную себестоимость товаров — это важно для корректных отчетов по прибыли.
:::

<img src="/img/quick-start/qs-products.png" alt="Товары в Excel" width="600" />
2. Перейдите в раздел **Товары**.
3. Нажмите на три точки в правом верхнем углу.
4. Выберите **Импорт**.
<img src="/img/quick-start/qs-import-1.png" alt="Начало импорта товаров в iBox" width="600" />
5. Перетащите файл в окно импорта или выберите его на компьютере.
6. Сопоставьте столбцы из файла с полями системы.
<img src="/img/quick-start/qs-import-2.png" alt="Импорт товаров в iBox" width="600" />
7. Нажмите **Начать импорт**.


### Добавление товаров вручную

1. Перейдите в раздел **Товары** и нажмите **плюс** в правом верхнем углу.
2. Заполните основные данные:
   - **Название товара**
   - **Единицу измерения**
   - **Артикул**

3. Во вкладке **Начальный остаток** укажите остатки по каждому складу:
   - **Склад**
   - **Количество**
   - **Цена**
   - **Дата**

4. Во вкладке **Прочее** при необходимости укажите:
   - **Бренд**
   - **Группу товара**

5. Нажмите **Сохранить**.

:::tip Совет  
Укажите **минимальный остаток товара**, чтобы система вовремя показывала дефицит на складе.
:::

## Прайс-листы

Прайс-листы позволяют работать с разными ценами — например, для разных клиентов или типов продаж.

### Создание прайс-листа

1. Перейдите в **Товары** → **Прайс-листы**.
2. Нажмите на плюс в правом верхнем углу.
3. Укажите название прайс-листа.
4. Выберите область применения:
    - Продажи  
    - Закупки  
5. Заполните цены товаров и укажите валюту.
<img src="/img/quick-start/qs-pricelist.png" alt="Создание прайс листа в iBox" width="600" />
6. Нажмите **Сохранить**.
