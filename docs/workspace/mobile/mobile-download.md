# Как скачать мобильное приложение iBox

Мобильное приложение iBox позволяет работать с системой с телефона: создавать заказы, управлять товарами и работать с клиентами в любое время.

## 📱 Android

Приложение для Android распространяется через Telegram-канал.

### Как скачать

1. Перейдите в Telegram-канал:  
   https://t.me/ibox_app  
2. Найдите файл с названием **app-release.apk**.  
3. Скачайте файл на устройство.  
4. После завершения загрузки откройте файл и установите приложение.

:::warning Важно
Если вы устанавливаете приложение впервые, система может запросить разрешение на установку из неизвестных источников. Разрешите установку в настройках устройства.
:::

---

## 🍎 iOS (iPhone)

Приложение доступно в App Store.

### Как скачать

1. Откройте **App Store**.  
2. Введите в поиске **ibox.io**.  
3. Найдите приложение **iBox**.  
4. Нажмите **Скачать**.

Или перейдите по прямой ссылке:  
https://apps.apple.com/uz/app/ibox-io/id1663454152

:::
