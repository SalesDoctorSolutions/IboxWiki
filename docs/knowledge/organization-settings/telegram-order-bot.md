# Телеграм-бот для заказов

## Для чего нужен телеграм-бот
С помощью телеграм-бота клиенты могут:
- оформлять заказы;
- просматривать баланс, баланс кешбека и акт сверки;
- получать уведомления о задолженности;
- получать уведомления о новых документах на своё имя.

---

## Как подключить телеграм-бота для заказов

Сначала нужно создать бота в **Telegram**:

1. В приложении Telegram найдите **@BotFather**.
2. Откройте мини-приложение.
3. Нажмите **Создать нового бота**.
4. Заполните данные:
   - название;
   - описание;
   - username с окончанием **bot**;
   - добавьте фото.
5. Нажмите **Создать**.
6. Скопируйте токен бота.

Далее в системе **iBox**:

1. Нажмите **Настройки** на левой панели.
2. Перейдите в раздел **Телеграм-бот** в настройках организации.
3. Откройте вкладку **Бот для контрагентов**.
4. Вставьте скопированный токен в поле **Токен бота**.
5. Нажмите **Далее**.
6. Выберите, кто может пользоваться ботом:
   - **Все** — новые пользователи могут зарегистрироваться через бота;
   - **Только клиенты системы** — доступ только у клиентов с указанным Telegram номером;
   - **Выбранные клиенты** — доступ только у выбранной группы клиентов.
7. Откройте вкладку **Настройка телеграм-заказов**.
8. Поставьте галочку **Включить**.
9. Выберите прайс-лист и склад.
10. Нажмите **Сохранить**.

---

## Настройки уведомлений

Можно настроить, какие уведомления будет получать клиент:
-	О новых документах
-	Срок оплаты по балансу клиента
-	Срок оплаты по отгрузкам

Для этого:

1. Нажмите **Настройки** на левой панели.
2. Перейдите в раздел **Телеграм-бот** в настройках организации.
3. Откройте вкладку **Бот для контрагентов**.
4. Перейдите в раздел **Уведомления**.
5. Откройте нужную вкладку.
6. Выберите необходимые поля.
7. Нажмите **Сохранить**.
