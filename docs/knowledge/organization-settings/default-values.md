# Значения по умолчанию

Чтобы не вводить одни и те же данные при создании каждого документа, можно задать значения по умолчанию.  
Тогда нужные поля будут заполняться автоматически, а при необходимости их можно изменить вручную.

## Как задать значения по умолчанию

1. Внизу боковой панели нажмите **Настройки**.
2. В разделе **Настройки организации** нажмите **Настройки**.
3. Откройте пункт **Значения по умолчанию**.
4. Нажмите на иконку **карандаша** справа от нужного значения.
5. Укажите нужное значение.
6. Нажмите **синюю галочку**, чтобы подтвердить действие,  
   или **красный крестик**, чтобы отменить.

---

## Какие значения можно задать по умолчанию

- **Валюта по умолчанию** — валюта, которую вы чаще используете в документах.
- **Скла по умолчанию** — основной склад для закупок и отгрузок.
- **Касса по умолчанию** — касса для приема оплат от клиентов.
- **Цена продажи по умолчанию** — прайс-лист для основной части продаж.
- **Цена межорг. отгрузок по умолчанию** — прайс-лист для отгрузок другим организациям.
- **Цена закупки по умолчанию** — прайс-лист, который чаще используется в закупках.
- **Единица хранения по умолчанию** — основная единица измерения товаров.
- **Резервирование заказа по умолчанию** — резервировать ли товар при создании заказа.
- **Вид акта сверки по умолчанию** — формат отображения акта сверки  
  (по документам, товарам или датам).
- **Режим скидки по умолчанию** — указывать скидку в сумме или в процентах.
- **Режим направления скидки по умолчанию** — использовать скидки или наценки.
- **Дата по умолчанию в списке документов** — фильтр по дате, применяемый автоматически.
- **Статусы документов по умолчанию** — статус, с которым создаются новые документы.
- **Период консигнации по умолчанию** — срок оплаты после отгрузки в долг.
- **Время на логистик по умолчанию** — среднее время доставки товара на склад.
- **Префикс телефонов по умолчанию** — код страны для контактов контрагентов.
- **Минимальная и максимальная сумма заказа** — ограничение суммы заказа.
- **Карта** — карта для просмотра адресов (Yandex, Google, 2GIS).
- **Тип штрихкода по умолчанию** — тип штрихкода при создании товаров.
