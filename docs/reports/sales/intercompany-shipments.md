# Межорганизационные отгрузки

Отчет **Межорганизационные отгрузки** позволяет просматривать список товаров, переданных между организациями.

В отчете доступны две вкладки: **Отправки товара** и **Приемки товара**.

## Отправки товара

Во вкладке **Отправки товара** отображается список товаров, которые вы передали другой организации.

### Как открыть вкладку

1. Откройте раздел **Отчеты** в левой панели.
2. В блоке **Отчеты по продажам** выберите **Межорганизационные отгрузки**.
3. Откройте вкладку **Отправки товара**.

### Доступные поля

- **Дата** — дата документа  
- **Номер** — номер документа  
- **Сумма** — сумма товара  
- **Получатель** — организация, принявшая товар  
- **Название товара** — наименование переданного товара  
- **Количество** — количество переданного товара  
- **Цена** — цена за единицу товара  

## Приемки товара

Во вкладке **Приемки товара** отображается список товаров, которые вы приняли от другой организации.

### Как открыть вкладку

1. Откройте раздел **Отчеты** в левой панели.
2. В блоке **Отчеты по продажам** выберите **Межорганизационные отгрузки**.
3. Откройте вкладку **Приемки товара**.

### Доступные поля

- **Дата** — дата документа  
- **Номер** — номер документа  
- **Сумма** — сумма товара  
- **Отправитель** — организация, передавшая товар  
- **Название товара** — наименование полученного товара  
- **Количество** — количество полученного товара  
- **Цена** — цена за единицу товара
