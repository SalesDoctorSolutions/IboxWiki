# Использовать несколько складов для закупки

Данная функция позволяет использовать **несколько складов в одном документе закупки или заказа поставщику**.

Это означает, что товары из одного документа можно распределять сразу по разным складам.

---

## Когда полезно использовать функцию

Использование функции помогает:

- оформлять одну закупку сразу для нескольких складов;
- экономить время при работе с поставщиками;
- корректно распределять товары по складам при поступлении;
- упрощать планирование запасов;
- уменьшать количество документов закупки.

Это особенно полезно, если компания имеет **несколько складов или торговых точек**, и поставщик привозит товары для разных складов одновременно.

---

## Как включить функцию

1. Перейдите в **Настройки** на левой панели.
2. В настройках организации откройте раздел **Настройки**.
3. Перейдите в раздел **Функционал**.
4. Найдите параметр **«Несколько складов для закупки»** и нажмите на иконку **карандаша** справа.
5. Выберите опцию **Да**.
6. Нажмите на **синюю иконку галочки**, чтобы сохранить изменения.

---

## Как создать закупку на несколько складов в одном документе

1. Перейдите в раздел **Закупки** или **Заказы поставщикам**.
2. Создайте новый документ.
3. В таблице товаров в колонке **«Доступно»** появится **название склада поступления** для каждого товара.
4. Нажмите на название склада в строке нужного товара.
5. Выберите склад из списка.
6. Повторите действие для других товаров при необходимости.
7. Нажмите **Сохранить**.

После сохранения товары будут **распределены по выбранным складам**.
