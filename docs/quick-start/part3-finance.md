# Часть 3. Клиенты, поставщики и балансы

В этом разделе вы добавите контрагентов и внесете стартовые финансовые данные.

## Клиенты и поставщики

Контрагенты — это клиенты и поставщики, с которыми вы работаете.


### Загрузка клиентов и поставщиков из Excel

1. Подготовьте файл Excel со следующими полями:
    - Название контрагента  
    - Номер телефона  
    - Баланс  
2. Перейдите в раздел **Клиенты**.
3. Нажмите на три точки в правом верхнем углу.
4. Выберите **Импорт**.
5. Загрузите файл.
6. Сопоставьте столбцы с полями системы.
7. Нажмите **Начать импорт**.

:::info Информация  
Чтобы загрузить поставщиков, добавьте в Excel колонку **Поставщик** и укажите значение `1` для нужных контрагентов.
:::

### Добавление клиентов вручную

1. Перейдите в раздел **Клиенты**.
2. Нажмите на плюс в правом верхнем углу.
3. Укажите:
    - Название  
    - Номер телефона  
4. Нажмите **Сохранить**.

### Добавление поставщиков вручную

1. Перейдите в раздел **Поставщики**.
2. Нажмите на плюс в правом верхнем углу.
3. Укажите:
   - Название  
   - Номер телефона  
4. Нажмите **Сохранить**.

## Начальные балансы

Чтобы корректно вести финансовый учет, внесите стартовые остатки денег и долгов.

### Корректировка баланса касс

1. Перейдите в **Деньги** → **Корректировки баланса**.
2. Нажмите на плюс в правом верхнем углу.
3. Выберите **Корректировка баланса кассы**.
4. Укажите кассу.
5. Заполните сумму и валюту.
6. Нажмите **Сохранить**.

### Корректировка баланса контрагентов

Используйте эту корректировку, если на момент запуска системы есть долги.

1. Перейдите в **Деньги** → **Корректировки баланса**.
2. Нажмите на плюс и выберите **Баланс контрагента**.
3. Укажите:
- Контрагента  
- Валюту и сумму  
   - Отрицательная сумма — клиент должен вам  
   - Положительная сумма — вы должны контрагенту  
- Комментарий — например, Стартовый долг  
4. Нажмите **Сохранить**.

Баланс контрагента будет учитываться во всех финансовых отчетах.

:::warning Предупреждение 
Не используйте корректировки баланса для приема оплаты, так как это может привести к ошибкам в рассчетах. Используйте данный модуль только для ввода начального баланса и внесения правок в рассчетах
:::
