# Корректировки баланса

Корректировки баланса помогают привести учет в порядок, если данные в системе не совпадают с реальными.

## Корректировка баланса кассы

Чтобы посмотреть список документов, связанных с корректировкой баланса кассы:

1. Перейдите в раздел **Деньги → Корректировки баланса**.
2. Откройте вкладку **Корректировки баланса кассы**.

### Когда использовать корректировку баланса кассы
- При внесении начальных остатков в кассах в систему.
- При сверке с бухгалтерией, чтобы привести фактический остаток в соответствие с документами.

### Как оформить корректировку баланса кассы
1. Перейдите в раздел **Деньги → Корректировки баланса**.
2. Нажмите на **плюс** в правом верхнем углу.
3. Выберите опцию **Корректировка баланса кассы**.
4. Заполните данные:
   - касса;
   - сумма корректировки;
   - валюта.
5. Нажмите **Сохранить**.

:::tip **Совет:**  
Нажмите на иконку калькулятора при заполнении суммы корректировки и введите, какой должен быть баланс после корректировки — система автоматически рассчитает сумму корректировки.
:::
---

## Корректировка баланса контрагента

Чтобы посмотреть список документов, связанных с корректировкой баланса контрагента:

1. Перейдите в раздел **Деньги → Корректировки баланса**.
2. Откройте вкладку **Корректировки баланса контрагента**.

### Когда использовать корректировку баланса контрагента
- При внесении начальных балансов контрагентов в систему.
- Для исправления несоответствий между реальным балансом контрагента и системой.
- Для списания долга контрагента.

:::warning **Важно:**  
Не используйте корректировки баланса для внесения оплат от контрагентов или контрагентам — это может привести к ошибкам в расчетах.
:::

### Как оформить корректировку баланса контрагента
1. Перейдите в раздел **Деньги → Корректировки баланса**.
2. Нажмите на **плюс** в правом верхнем углу.
3. Выберите опцию **Баланс контрагента** (при необходимости можно выбрать несколько).
4. Заполните данные:
   - касса;
   - сумма корректировки;
   - валюта.
5. Нажмите **Сохранить**.
