Техкарта — это шаблон сборки, в котором заранее задан состав продукции и пропорции сырья.

Используйте техкарты, чтобы быстро создавать сборки без ручного ввода и снизить количество ошибок.

## Когда полезно использовать техкарты

Техкарты особенно полезны, если:

- вы регулярно производите одинаковую продукцию;
- важно соблюдать точные пропорции сырья;
- нужно ускорить создание сборок;
- сотрудники выполняют типовые операции;
- требуется стандартизировать производство.

## Особенности

- система автоматически пересчитывает расход сырья при изменении объема выпуска;
- техкарты можно использовать многократно;
- создание техкарт не обязательно, но упрощает работу при регулярных процессах.

## Как создать техкарту

1. Откройте **Фабрика → Техкарты**.  
2. Нажмите **Плюс**.  
3. Укажите название техкарты.  
4. В блоке **Готовая продукция**:
   - выберите товар;
   - укажите количество выхода.
5. В блоке **Сырье** добавьте компоненты:
   - выберите товары из списка;
   - укажите количество для каждого компонента.
6. При необходимости заполните поле **Комментарий**.  
7. Нажмите **Сохранить**.

---

:::info
При необходимости вы можете указать долю себестоимости для каждого товара в блоке **Готовая продукция**. Это позволяет точнее распределять себестоимость.
:::

---

После сохранения техкарта будет доступна при создании сборки.

---

:::tip Совет
Используйте кнопку **Импорт из Excel**, чтобы быстро загрузить состав техкарты.
:::

---

## Как использовать техкарту

При создании сборки:

- выберите техкарту;
- укажите объем производства;
- система автоматически подставит сырье и рассчитает его количество.

---

:::info Информация
Вы можете редактировать техкарты в любое время. Изменения применяются только к новым сборкам.
:::
