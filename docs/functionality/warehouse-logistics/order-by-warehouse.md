# Автоматически разделять заказы по складам

Данная функция позволяет **автоматически разделять документ заказа по каждому складу**.

Если при создании заказа в списке товаров указаны **разные склады**, то при сохранении система создаст **несколько отдельных заказов** — по одному для каждого склада со своим списком товаров.

---

:::info Информация
Данная функция работает **только при создании нового документа** и не применяется при редактировании уже созданного заказа.
:::

---

## Когда полезно использовать функцию

Использование функции помогает:

- автоматически распределять заказы по складам без ручного разделения;
- упрощать дальнейшее создание отгрузок;
- уменьшать вероятность ошибок при обработке заказов;
- ускорять работу сотрудников при оформлении документов;
- улучшать контроль резервирования товаров по складам.

Это особенно полезно, если:

- компания использует несколько складов;
- товары одного заказа должны отгружаться с разных складов;
- необходимо контролировать загрузку каждого склада;
- обработкой заказов занимаются разные сотрудники.

---

## Как включить функцию

1. Перейдите в **Настройки** на левой панели.
2. В настройках организации откройте раздел **Настройки**.
3. Перейдите в раздел **Функционал**.
4. Найдите параметр **«Автоматически разделять заказы по складам»** и нажмите на иконку **карандаша** справа.
5. Выберите опцию **Да**.
6. Нажмите на **синюю иконку галочки**, чтобы сохранить изменения.
