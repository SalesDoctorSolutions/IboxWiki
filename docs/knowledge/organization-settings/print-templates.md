# Шаблоны для печати

В iBOX есть возможность создавать и использовать собственные шаблоны для печати накладных.

---

## Как добавить свой шаблон

1. Нажмите **Настройки** в нижней части левой панели.
2. В настройках организации перейдите в раздел **Шаблоны для печати**.
3. Скачайте системный шаблон для нужного документа, нажав на иконку скачивания.
4. Скачайте подходящий базовый шаблон.
5. Настройте шаблон в Excel:
   - отредактируйте системный шаблон;
   - добавьте нужные поля и логотип;
   - используйте системные поля из базового шаблона.
6. В разделе **Шаблоны для печати** нажмите **+ Загрузить шаблон**.
7. Укажите название шаблона.
8. Выберите Excel-файл с вашей накладной.
9. При необходимости укажите ширину печати (по умолчанию — A4).
10. Нажмите **Сохранить**.

---

## Как выбрать основной шаблон

1. Перейдите в **Настройки → Шаблоны для печати**.
2. Нажмите на три точки справа от нужного шаблона.
3. Выберите **Сделать основным**.

---

## Как настроить дополнительные параметры

1. Перейдите в **Настройки → Шаблоны для печати**.
2. Нажмите на три точки справа от нужного шаблона.
3. Выберите необходимые параметры.
4. Нажмите **Сохранить**.

### Какие дополнительные параметры можно настроить

#### Сортировка товаров
Определяет порядок отображения товаров в накладной:
- по алфавиту;
- по артикулу;
- по категориям;
- по группам;
- по брендам;
- по складам.

#### Объединять одинаковые товары
Если товар встречается в документе несколько раз — объединяет дубликаты в одну строку.

### Дополнительные параметры для заказов

- **Показывать только неотгруженные товары** — отгруженные позиции не будут отображаться при печати.

### Дополнительные параметры для отгрузок

- **Разделять отгрузку по складам** — печатает отдельную накладную для каждого склада.
- **Отображать чистое количество (с учётом возвратов)** — возвращенные товары не будут отображаться в печатной форме.
