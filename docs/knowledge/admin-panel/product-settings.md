# Настройка товаров и цен в организациях

## Как добавить товары в организации
1. Перейдите в **панель администратора**.
2. Откройте раздел **Товары → Товары**.
3. Выберите товары, которые хотите добавить в организации.
4. Нажмите на **три точки** на верхней панели.
5. Выберите опцию **Добавить в организацию**.
6. Отметьте организации, в которые хотите добавить товары.
7. Нажмите **Применить**.

---

## Как убрать товары из организаций
1. Перейдите в **панель администратора**.
2. Откройте раздел **Товары → Товары**.
3. Выберите товары, которые хотите убрать из организаций.
4. Нажмите на **три точки** на верхней панели.
5. Выберите опцию **Удалить из организации**.
6. Отметьте организации, из которых нужно убрать товары.
7. Нажмите **Применить**.

---

## Как создавать товары сразу во всех организациях

Если вы хотите, чтобы при создании товара в одной организации он автоматически появлялся во всех остальных:

1. Перейдите в **панель администратора**.
2. Откройте раздел **Настройки → Глобальные настройки**.
3. Поставьте галочку **«Автоматически прикреплять новые товары ко всем организациям»**.
4. Нажмите **Сохранить**.

Если вы, наоборот, не хотите, чтобы товары автоматически появлялись во всех организациях, просто снимите эту галочку.

---

## Как добавить или убрать прайс-листы из организаций
1. Перейдите в **панель администратора**.
2. Откройте раздел **Товары → Прайс-листы**.
3. Нажмите на **три точки** рядом с нужным прайс-листом.
4. Выберите опцию **Настроить организации**.
5. Отметьте организации, которые будут использовать этот прайс-лист.
6. Нажмите **Сохранить**.

---

## Для чего нужна вторая единица хранения и как её добавить

Вторая единица хранения нужна, когда кроме основной единицы хранения, на которую устанавливается цена, вам также нужно учитывать дополнительную единицу, не связанную с основной через коэффициент.  
Вторая единица вводится вручную при каждой операции.

### Пример
Вы продаёте плиточный клей.

- Основная единица хранения, по которой задаётся цена — **кг**.
- Вторая единица хранения — **ведро**.

При этом в каждом ведре может быть разное количество клея.

### Как будут выглядеть операции

1. Вы продаёте 5 кг клея на развес  
   В программе указываете: **5 кг и 0 ведер**.

2. Затем вы продаёте ведро клея, в котором осталось 12 кг  
   В программе указываете: **12 кг и 1 ведро**.

Стоимость при этом всегда рассчитывается по количеству **кг**.

### Как включить использование второй единицы хранения

1. Перейдите в **панель администратора**.
2. Откройте раздел **Настройки → Глобальные настройки**.
3. Поставьте галочку **«Использовать вторую единицу хранения»**.
4. Нажмите **Сохранить**.

:::info **Информация:**  
Вторую единицу хранения товара можно указать при его создании или редактировании.
:::
