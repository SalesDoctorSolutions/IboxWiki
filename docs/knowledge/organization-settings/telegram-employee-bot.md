# Телеграм-бот для сотрудников

## Для чего нужен телеграм-бот
С помощью этого бота сотрудники могут получать уведомления в Telegram-группу о:
- создании новых документов (заказ, отгрузка, оплата);
- редактировании документов;
- удалении документов;
- неуплате долга клиентами.

---

## Как подключить телеграм-бота для сотрудников

Сначала нужно создать бота в **Telegram**:

1. В приложении Telegram найдите **@BotFather**.
2. Откройте мини-приложение.
3. Нажмите **Создать нового бота**.
4. Заполните данные:
   - название;
   - описание;
   - username с окончанием **bot**;
   - добавьте фото.
5. Нажмите **Создать**.
6. Скопируйте токен бота.
7. Добавьте созданного бота в Telegram-группу с сотрудниками.

Далее в системе **iBOX**:

1. Нажмите **Настройки** на левой панели.
2. Перейдите в раздел **Телеграм-бот** в настройках организации.
3. Откройте вкладку **Бот для сотрудников**.
4. Вставьте скопированный токен в поле **Токен бота**.
5. Нажмите **Далее**.
6. Выберите, какие уведомления будут получать сотрудники:
   - откройте вкладку нужного типа уведомлений;
   - нажмите **Включить**;
   - настройте дополнительные поля;
   - выберите группы, куда отправлять уведомления  
     (при необходимости добавьте условия).
7. Нажмите **Сохранить**.
